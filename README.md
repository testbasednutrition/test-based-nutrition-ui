# TestBased Nutrition - UI Codebase

This repository contains the React frontend codebase for the TestBased Nutrition website.

## Tech Stack
- **Framework**: React / Vite
- **Styling**: Tailwind CSS
- **Components**: shadcn/ui
- **Routing**: React Router

## Getting Started

First, install the dependencies:
```bash
npm install
```

Then, run the development server:
```bash
npm run dev
```

Open [http://localhost:8080](http://localhost:8080) with your browser to see the result.

## Core Features Implemented

*   **Dynamic 3D Gallery "What We Treat" Section**: Interactive 3D flip cards for service categories (Women's Health, Men's Health, Children's Health, Neurodivergence, Skin Health, Sports Performance) that engage users and reveal detailed treatment information on hover.
*   **Director-Led & Expert-Governed Bios**: A beautifully framed grid presenting the medical team and leadership (e.g., Dr. Ishtiaq Rehman, Natasha Sundharawipata, Neil Parsley) to establish immediate trust and authority.
*   **Methodology & Pillars Breakdown**: Clear, animated sections explaining "The Foundations of Test-Based Nutrition" with distinct typography ensuring the scientific approach is easily digestible.
*   **"Partner With Us" Integration**: Distinct pathways and landing pages designed specifically for B2B expansion, detailing how clinics, pharmacies, and health clubs can collaborate and integrate the diagnostic services.
*   **Refined Global Design System**: Consistent typography (elegant *Playfair* serif headings paired with clean *Montserrat* sans-serif body text) and a cohesive color palette (signature red against transparent/cream backgrounds) applied universally, including global button hover states.
*   **Responsive & Fast Architecture**: Built on Vite and React with Tailwind CSS, ensuring lightning-fast load times and perfect rendering across mobile, tablet, and desktop devices.

## Business Benefits

1.  **Enhanced User Engagement**: Micro-animations and premium UI elements (like the 3D card flips) keep visitors actively exploring the site longer, reducing bounce rates.
2.  **Elevated Brand Trust**: By spotlighting the expert medical directors and clearly articulating the testing methodology with a clean, high-end aesthetic, prospective patients feel confident in the services.
3.  **Clear Conversion Pathways**: Strategically placed, highly visible calls-to-action (e.g., "Book Now", "Partner With Us") guide both B2C patients and B2B partners smoothly toward contacting the team.
4.  **Scalable UI Foundation**: Utilizing a modular component architecture (`shadcn/ui`) means future pages, diagnostic tests, or partner dashboard modules can be added rapidly while strictly maintaining the brand's premium look and feel.
